Тема проекта - поликлинника
Проект работает в трех режимах - режим администратора, режим регистратуры и режим врача
Режим регистратуры отличается от режима администратора только кнопкой "Редактировать сотрудников"
Логин/пароль админа - admin/admin
По кнопке "Редактировать сотрудников" снизу можно будет посмотреть другие логины и пароли
В проекте сильно не хватает фильтрации данных, так любой врач сможет редактировать заметки другого врача, также из-за этого не удалось сделать удобнй интерфейс, отображающий информацию
об отдельно взятом пользователе, все навалено в кучу, но знаний на такое не хватает.
По той же причине не удалось сделать вывод отчетов по датам, в остальном, постарались максимально сохранить архитектуру
